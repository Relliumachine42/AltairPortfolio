About Me

I am a software developer based out of North Carolina, and this site shows off some applications and websites that I've built. By using C# and .NET to build web applications, I can turn ideas into practical and attractive software products. I'm hoping we can work together on your next project!
Download Resume

What I Use

I use C# on a daily basis to build full-stack web applications. You can see my work implementing C# in my Projects section!

I use proven design patterns and frameworks to build websites. ASP.NET Core Web MVC is the appropriate choice for most projects.

I use HTML, CSS, and Bootstrap to build beautiful, responsive websites. In my work, front-end design is prioritized alongside the back-end code.

Since JavaScript is the language of the web, I use vanilla JS and am confortable with many JS frameworks. Check out my challenges section for some JS projects!

SQL is the language of the database. My projects all utilize a database. I have used Postgres on all of them. I can write SQL on any DB platform.

I use git and github for all my projects. Git is the most used source control framework on the planet. I also employ CI/CD for publishing.
